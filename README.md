# Ericotour Safari Website

This is a fully responsive, mobile-first website for a fictional tour company, **Ericotour Safari**, built with only HTML, CSS, and JavaScript — no backend or database required.

### 🌍 Features

- Homepage with animated hero section
- Booking form (submits to email via [Formsubmit](https://formsubmit.co))
- Contact form with embedded map and WhatsApp link
- Light/Dark mode toggle
- Mobile nav menu
- Smooth scroll animations
- Clean, safari + oceanic themed layout
- Social media links in footer

### 🛠 Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Formsubmit for email delivery
